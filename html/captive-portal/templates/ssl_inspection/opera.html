This network employs a technology called captive portal which allows the administrator(s) to restrict network access to trusted users. In order to prevent certificate security warnings when accessing this portal, the network's filtering unit must issue to your computer or mobile device a file called an SSL certificate.

This guide will help you install the SSL filtering certificate for the Opera browser.

Note: This guide is for Opera version 12 and below. Newer versions of Opera will use your computer system's certificate or keychain manager instead of your browser's local certificate manager. If you have Opera version 15 or above, please download either the Mac OS X or Windows based certificate, or any of the other platforms, depending on your system.

Step 1

First, click the Download Certificate button below; this will download the certificate file to a temporary location on your computer.

Download Certificate

Step 2

Next, a dialogue will appear, asking if you would like to install the certificate authority. Click Install.

(Note: These images show Opera on Mac OS X. If your computer runs Windows or Linux, the buttons may look different - but the words should be the same.)

Step 3

After clicking Install, another dialogue will appear, asking if you are sure you want to trust this issuer. Click OK here.

Step 4

You're done! You may continue browsing as usual; in most cases, you will not notice anything different after the certificate is installed.
